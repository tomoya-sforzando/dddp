# Dic Deeper into Data Pipeline

## About Data Pipeline

データパイプラインとは，データ処理を行なう小さなタスク（1回のファイルコピーや，SQLの実行など）を順次実行することにより，最終的に求める結果を得るための一連のプロセス

### 狭義

単体のシステム内で完結するパイプライン  
（SparkやGoogle Cloud Dataflow，など⁠）

### 広義

複数のシステムを組み合わせて大きなパイプライン  
→ ワークフロー管理が必要
